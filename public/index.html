What is Bridges?

Bridges is an open-source software package to make learning data structures more interesting:
  • Built-in connections to real-world datasets reduces the overhead of reading and parsing datafiles
  • Visualization capabilities so you can debug, view, and share your data structures on the web
Interested? Check out our documentation or head over to our user gallery
